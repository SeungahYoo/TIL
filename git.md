### Git

- working copy

  수정된 파일 목록

- add

  working copy에서 check를 하는 행위

- index, staging area

  working copy에 있는 변경된 파일을 add하면 index / staging area 라는 공간으로 이동.

- commit

  index / staging area에 있는 파일들을 commit 하면 version log에 들어간다. 하나의 version이 된다.

- repository 저장소

  각각의 version이 저장되어 있는 곳.
